# Facebok 2.0

## In development

This social web app is created in the latest Symfony 5 with all useful tools:

* **Symfony 5**
* **React**
* Bootstrap
* Sass Loader 
* **Docker**
* composer
* PHPUnit
* php-cs-fixer
* phpstan
* eslint
* webpack

**Usage**
---

```
sudo docker-compose up
yarn encore dev --watch
```

**Installation Options**
---

1. Install with `composer`
    + `$ composer install`
    + `$ yarn install`
